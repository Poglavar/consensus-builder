In this branch we will work on the blockchain portion of the project.

The feature will consist of two parts:
1) The smart contracts, in the blockchain folder.
2) Changes to the frontend and backend to integrate with the smart contracts.

The smart contracts will be in Solidity and target EVM-compatible chains.
The frontend will integrate with EVM-compatible wallets using a toolkit that supports multiple wallets.

We'll rely on Scaffold ETH for inspiration as to which components are modern and how to wire them together. But we don't need everything they have, the goal is simplicity, light weight and using only what we need.
https://scaffoldeth.io/
https://github.com/scaffold-eth/scaffold-eth-2

The idea is that blockchain will store and serve the most important backend data of the application. The data in the browser will use blockchain as source of truth, and the blockchain data will be usable by other frontends.

---------------
Smart contracts
---------------

These Urban Game Theory concepts are blockchain objects:
- parcel
- proposal
- owner

Parcel:
- All land on the map should/will be parcellized, that is, no surface area that is not part of a parcel. We will deal with this in a script called mint-parcels.js
- The parcels are minted as ERC721 NFTs, owned by different addresses, where the admin can change the owner. For purposes of the demo, they will all be assigned as owners to addresses generated by one HD wallet.

Proposals:
- Proposals are ERC721 NFTs, extended with our Proposal.sol capabilities
- If the user is connected with a wallet, each Create Proposal button in the UI mints a proposal NFT on the blockchain the Wallet is currently connected to.
- Each proposal NFT has an image, which is a thumbnail of the map with the parcels that are part of the proposal highlighted
- Each proposal NFT links to the parcel NFTs

- Or maybe have it link to geographies, independent of parcels?

Owners:
- When you log in with your wallet using the "Connect Wallet" button we search the database (or make a blockchain call) to get all the land NFTs your address owns. If some are found, a list of your land parcels is available as in the "Owned Parcels" section of your User Details.
